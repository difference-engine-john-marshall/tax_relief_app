# John Marshall Law Innocent Spouse Tax Relief App

# Feature list:
* Ruby Version 2.2.3
* Rails Version 4.2.5
* PostgreSQL 0.15
* RSpec 3.4.2
* Factory Girls 4.6

# Database
Create Database
```terminal
rake db:create
```
Migrate Database
```terminal
rake db:migrate
```
Seed Database
```terminal
rake db:seed
```

# TestSuite
To Run Tests
```terminal
rspec
```

# Install Features
To configure gems
```terminal
bundle
```
